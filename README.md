# better-date.js

## Utilisation & Example

> To install it, run the following command:

```batch
npm i better-date.js or yarn add better-date.js
```

> Code example:

```js
// don't forget to require the module!
const { date } = require("better-date.js");

/* examples */

console.log(new date());
```
